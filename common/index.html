Web #

  • WebSockets: WebSockets 是一种先进的技术。它可以在用户的浏览器和服务器之间打开交互式通信会话。使用此 API，您可以向服务器发送消息并接收事件驱动的响应，而无需通过轮询服务器的方式以获得响应。

框架与工具 #

Vite #

了解一下 Vite 吧，也许会替代 webpack 成为未来的趋势。

Vite 是一种新型的前端构建工具，提供热加载的开发服务器与构建命令，使用原生的 ES Module 打包。

  • 打包基于 Rollup
  • 开发服务器基于 Koa
